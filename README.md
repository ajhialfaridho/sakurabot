## CARA INSTALL
# TERMUX
```bash
> download termux
> buka
> pkg install git
> pkg install ffmpeg
> pkg install nodejs
> apt update && apt upgrade
> git clone https://github.com/ajhialfaridho/sakurabot
> cd sakurabot
> npm i -g cwebp && npm i -g ytdl
> npm i && npm i got
> bash install.sh
> node index.js
```
